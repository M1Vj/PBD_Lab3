# Part 3: Breakpoints

The project uses two main responsive breakpoints:

- `680px`: shifts the search form into two columns and upgrades the forecast to a 2-column grid
- `960px`: splits the results area into current-weather and forecast columns, and upgrades the forecast to a 5-column desktop grid

## Why these breakpoints were chosen

- `680px` covers common tablet widths and larger phones in landscape mode
- `960px` gives enough room for a true desktop presentation without making the cards too narrow
