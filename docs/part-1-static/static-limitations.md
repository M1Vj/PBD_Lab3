# Limitations of a Static Implementation
- **No live data**: The weather and forecast display is completely hardcoded. Users won't know the real temperature.
- **Manual content updates only**: Changing the city or weather condition requires modifying the source code.
- **No user-driven search behavior**: Submitting the form does nothing.
- **No real-time state changes**: The dashboard cannot respond to user interaction effectively (e.g., clicking to toggle theme does noting right now).
- **Limited scalability**: Every new city needs a new HTML block or component.
